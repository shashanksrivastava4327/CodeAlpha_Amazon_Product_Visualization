# 📊 Amazon Product Data Visualization

## Overview

This project analyzes and visualizes Amazon product data using Python. The goal is to transform raw data into meaningful visual insights that help understand product trends, customer ratings, product categories, and discount patterns.

The project demonstrates the use of data visualization techniques to explore and communicate insights effectively.

---

## Objectives

- Explore Amazon product dataset.
- Identify trends and patterns in product information.
- Visualize customer ratings and discount distributions.
- Present insights through clear and informative charts.

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code
- GitHub

---

## Dataset Features

The dataset contains information such as:

- Product Name
- Product Category
- Product Rating
- Rating Count
- Actual Price
- Discounted Price
- Discount Percentage
- Customer Reviews

---

## Visualizations

### Top Product Categories
Displays the most common product categories in the dataset.

### Rating Distribution
Shows how customer ratings are distributed across products.

### Discount Distribution
Analyzes discount percentages offered on products.

### Top Rated Products
Highlights products with the highest ratings.

---

## Key Findings

- Electronics and accessories dominate the dataset.
- Most products maintain high customer ratings.
- Large discounts are common across several categories.
- Highly rated products tend to attract significant customer engagement.

---

## How to Run

### Install Required Libraries

```bash
pip install pandas matplotlib seaborn
```

### Execute the Project

```bash
python visualization.py
```

The project generates the following visualizations:

These visualizations help understand product performance and customer preferences.
